    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Zed in minutes
  </p>
  
  <Steps>
    <Step title="Install Zed">
      Download Zed for your platform: macOS, Linux, or Windows. Install using your package manager or direct download from [zed.dev](https://zed.dev/download).
      
      ```bash macOS
      brew install zed
      ```
      
      ```bash Linux
      curl -f https://zed.dev/install.sh | sh
      ```
    </Step>
    
    <Step title="Open a project">
      Launch Zed and open a folder or project. You can also use the CLI to open files directly:
      
      ```bash
      zed path/to/your/project
      ```
      
      Zed automatically detects your project's language and installs the appropriate language server.
    </Step>
    
    <Step title="Start coding">
      Experience blazing-fast editing with GPU acceleration, intelligent code completion, and integrated AI assistance. Press `Cmd+K` (macOS) or `Ctrl+K` (Linux/Windows) to open the command palette and explore features.
    </Step>
  </Steps>
</div>
