# Quick Start

This guide will get you from zero to productive in Zed. You'll learn the essential commands, open your first project, and start editing code.

## Opening Your First Project

<Steps>
  <Step title="Open a Folder">
    Launch Zed and open a project folder using one of these methods:
    
    **From the command line:**
    ```bash
    zed ~/projects/my-app
    ```
    
    **From within Zed:**
    - Press `Cmd+O` (macOS) or `Ctrl+O` (Linux/Windows)
    - Navigate to your project folder
    - Click "Open"
  </Step>
  
  <Step title="Explore the Interface">
    When you open a project, you'll see:
    
    - **Project Panel** (left): Browse your project files
    - **Editor** (center): Edit your code
    - **Assistant Panel** (right): AI chat and tools (can be toggled)
    - **Status Bar** (bottom): Git status, diagnostics, and language info
  </Step>
  
  <Step title="Open a File">
    Use `Cmd+P` (macOS) or `Ctrl+P` (Linux/Windows) to open the file picker. Start typing a filename and press Enter to open it.
  </Step>
</Steps>

## Essential Keyboard Shortcuts

These are the most important shortcuts to memorize:

### Navigation

| Action              | macOS         | Linux/Windows  | Description                           |
| ------------------- | ------------- | -------------- | ------------------------------------- |
| **Command Palette** | `Cmd+Shift+P` | `Ctrl+Shift+P` | Access all commands                   |
| **Go to File**      | `Cmd+P`       | `Ctrl+P`       | Quickly open files by name            |
| **Go to Symbol**    | `Cmd+Shift+O` | `Ctrl+Shift+O` | Jump to functions, classes, etc.      |
| **Find in Project** | `Cmd+Shift+F` | `Ctrl+Shift+F` | Search across all files               |
| **Go to Line**      | `Ctrl+G`      | `Ctrl+G`       | Jump to a specific line number        |

### Editing

| Action                | macOS     | Linux/Windows | Description                        |
| --------------------- | --------- | ------------- | ---------------------------------- |
| **Multi-cursor**      | `Cmd+D`   | `Ctrl+D`      | Select next occurrence             |
| **Add cursor above**  | `Cmd+Alt+Up` | `Ctrl+Alt+Up` | Add cursor on line above        |
| **Add cursor below**  | `Cmd+Alt+Down` | `Ctrl+Alt+Down` | Add cursor on line below      |
| **Comment line**      | `Cmd+/`   | `Ctrl+/`      | Toggle line comment                |
| **Format document**   | `Cmd+Shift+I` | `Ctrl+Shift+I` | Format code using language formatter |

### Tools

| Action              | macOS         | Linux/Windows  | Description                  |
| ------------------- | ------------- | -------------- | ---------------------------- |
| **Toggle Terminal** | `` Ctrl+` ``  | `` Ctrl+` ``   | Open/close integrated terminal |
| **Open Settings**   | `Cmd+,`       | `Ctrl+,`       | Open settings editor         |
| **Agent Panel**     | `Cmd+Shift+A` | `Ctrl+Shift+A` | Open AI assistant            |
| **Extensions**      | `Cmd+Shift+X` | `Ctrl+Shift+X` | Browse and install extensions |

<Note>
Forgot a shortcut? Press `Cmd+Shift+P` (or `Ctrl+Shift+P`) to open the command palette and search for any command.
</Note>

## Using the Command Line Interface

The Zed CLI lets you open files and projects from your terminal.

### Opening Files and Folders

```bash
# Open current directory
zed .

# Open a specific file
zed README.md

# Open multiple files
zed src/main.rs src/lib.rs

# Open file at specific line
zed main.rs:42

# Open file at line and column
zed main.rs:42:10
```

### Common CLI Options

<CodeGroup>
```bash Wait for File to Close
zed --wait README.md
# Useful for git commit messages
export EDITOR="zed --wait"
```

```bash Open in New Window
zed --new ~/projects/another-project
```

```bash Add to Current Window
zed --add newfile.txt
```

```bash Compare Files
zed --diff old.txt new.txt
```
</CodeGroup>

See the full [CLI Reference](https://zed.dev/docs/reference/cli) for all available options.

## Your First Editing Session

Let's practice the essential editing features:

### 1. Multi-cursor Editing

<Steps>
  <Step title="Select a word">
    Double-click a word to select it.
  </Step>
  
  <Step title="Select next occurrence">
    Press `Cmd+D` (macOS) or `Ctrl+D` (Linux/Windows) to select the next occurrence of the same word.
  </Step>
  
  <Step title="Edit all at once">
    Start typing to edit all selected occurrences simultaneously.
  </Step>
</Steps>

### 2. Go to Definition

<Steps>
  <Step title="Place cursor on a symbol">
    Click on a function, variable, or class name.
  </Step>
  
  <Step title="Jump to definition">
    Press `F12` or right-click and select "Go to Definition".
  </Step>
  
  <Step title="Go back">
    Press `Ctrl+-` (macOS) or `Alt+Left` (Linux/Windows) to return to your previous location.
  </Step>
</Steps>

### 3. Find and Replace

```bash
# Find in current file
Cmd+F (macOS) or Ctrl+F (Linux/Windows)

# Find and replace in current file
Cmd+H (macOS) or Ctrl+H (Linux/Windows)

# Find in entire project
Cmd+Shift+F (macOS) or Ctrl+Shift+F (Linux/Windows)
```

### 4. Using the Integrated Terminal

<Steps>
  <Step title="Open terminal">
    Press `` Ctrl+` `` to open the integrated terminal.
  </Step>
  
  <Step title="Run commands">
    The terminal starts in your project directory. Run any command:
    
    ```bash
    npm install
    cargo build
    python manage.py runserver
    ```
  </Step>
  
  <Step title="Create additional terminals">
    Click the `+` icon in the terminal panel to open multiple terminal sessions.
  </Step>
</Steps>

## Setting Up Your Language

Zed includes built-in support for many languages. For others, you'll need to install an extension.

<Steps>
  <Step title="Open Extensions">
    Press `Cmd+Shift+X` (macOS) or `Ctrl+Shift+X` (Linux/Windows).
  </Step>
  
  <Step title="Search for your language">
    Type the name of your programming language (e.g., "Python", "Go", "Rust").
  </Step>
  
  <Step title="Install the extension">
    Click "Install" next to the language extension.
  </Step>
  
  <Step title="Verify language support">
    Open a file in that language and check that syntax highlighting and code intelligence work.
  </Step>
</Steps>

<Note>
Some languages require additional tools (like language servers) to be installed on your system. Check the [Languages](https://zed.dev/docs/languages) documentation for language-specific setup instructions.
</Note>

## Configuring Zed

Customize Zed to match your preferences:

### Open Settings

Press `Cmd+,` (macOS) or `Ctrl+,` (Linux/Windows) to open the Settings Editor. You can search for any setting and change it directly.

### Common Settings to Configure

<CodeGroup>
```json Theme
{
  "theme": "Gruvbox Dark",
  "ui_font_size": 16,
  "buffer_font_size": 14
}
```

```json Font
{
  "buffer_font_family": "Fira Code",
  "buffer_font_features": {
    "calt": true
  }
}
```

```json Format on Save
{
  "format_on_save": "on",
  "formatter": "language_server"
}
```

```json Vim Mode
{
  "vim_mode": true
}
```
</CodeGroup>

### Quick Theme Switching

Press `Cmd+K Cmd+T` (macOS) or `Ctrl+K Ctrl+T` (Linux/Windows) to open the theme picker and preview themes instantly.

## Trying AI Features

Zed includes built-in AI assistance to help you code faster.

### Agent Panel (AI Chat)

<Steps>
  <Step title="Open Agent Panel">
    Press `Cmd+Shift+A` (macOS) or `Ctrl+Shift+A` (Linux/Windows).
  </Step>
  
  <Step title="Ask a question">
    Type a question or request:
    - "How do I read a file in Python?"
    - "Explain this function"
    - "Write a test for this code"
  </Step>
  
  <Step title="Include context">
    Select code in your editor before opening the panel to automatically include it as context.
  </Step>
</Steps>

### Inline Assistant

<Steps>
  <Step title="Select code">
    Highlight the code you want to modify.
  </Step>
  
  <Step title="Invoke assistant">
    Press `Cmd+Enter` (macOS) or `Ctrl+Enter` (Linux/Windows).
  </Step>
  
  <Step title="Give instructions">
    Type what you want to do:
    - "Add error handling"
    - "Convert to async/await"
    - "Add comments"
  </Step>
</Steps>

<Note>
AI features require configuration of an LLM provider. See the [AI Configuration](https://zed.dev/docs/ai/configuration) guide to set up Claude, GPT-4, Gemini, or other providers.
</Note>

## Working with Git

Zed has built-in Git integration:

### View Changes

- **Gutter indicators**: Modified lines show in the editor gutter
- **Git Panel**: Press `Cmd+Shift+G` (macOS) or `Ctrl+Shift+G` (Linux/Windows)
- **Inline blame**: See who changed each line

### Stage and Commit

<Steps>
  <Step title="Open Git panel">
    Press `Cmd+Shift+G` (macOS) or `Ctrl+Shift+G` (Linux/Windows).
  </Step>
  
  <Step title="Stage changes">
    Click files to stage them, or click "Stage All".
  </Step>
  
  <Step title="Write commit message">
    Type your commit message in the text box.
  </Step>
  
  <Step title="Commit">
    Press `Cmd+Enter` (macOS) or `Ctrl+Enter` (Linux/Windows) to commit.
  </Step>
</Steps>

### View Diffs

Click on a modified file in the Git panel to see a side-by-side diff.
